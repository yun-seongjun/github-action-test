# 프로젝트 코드 리뷰 가이드

## 코드 스타일
- TypeScript 사용 시 strict 모드 필수
- 함수는 40줄 이하로 유지
- 변수명은 camelCase 사용

## 금지 사항
- any 타입 사용 금지
- console.log 커밋 금지
- 하드코딩된 비밀값 금지

## PR 리뷰 체크리스트
1. 코드 스타일 가이드 준수
2. 의미있는 변수 이름 사용
3. DRY 원칙 준수
4. KISS 원칙 준수
5. YAGNI 원칙 준수
