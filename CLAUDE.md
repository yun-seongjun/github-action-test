# 프로젝트 코드 리뷰 가이드

## 참조 문서
코드 리뷰 시 아래 Confluence 문서를 반드시 참조하세요.

| 페이지 ID | 문서명 | 용도         |
|-----------|--------|------------|
| 484082682 | FE - Style Guide 정의 | 코드 컨벤션 정의1 |
| 504987723 | Style Guide | 코드 컨벤션 정의2 |

`confluence_get_page` 도구로 위 문서들을 조회한 후 리뷰를 진행하세요.

## PR 리뷰 체크리스트
1. 코드 스타일 가이드 준수
2. 의미있는 변수 이름 사용
3. DRY 원칙 준수
4. KISS 원칙 준수
5. YAGNI 원칙 준수
